Create Test Paper

Create a paper first, then attach questions to it from the question bank and publish only when ready.

Custom test: publish with at least 1 question.
Fixed start means students can begin only after the configured start time. If they start late, available remaining time may be shorter.
Time window means students may start any time within the allowed window, but once started they must finish within the valid remaining duration.
Once a student starts the test, the timer continues on the server and the attempt will auto-submit when time ends.
