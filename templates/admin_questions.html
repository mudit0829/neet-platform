Bulk Upload Questions

Paste a JSON array of question objects and upload many questions in a single submission.

Add Single Question

Required keys

Each question object should include these fields.

  • subject
  • chapter
  • difficulty_level
  • stem
  • option_a, option_b, option_c, option_d
  • correct_option
  • explanation

Important checks

  • Subject name must already exist in the database.
  • Chapter name must already exist under that subject.
  • correct_option must be A, B, C, or D.
  • difficulty_level should be easy, medium, or hard.

Sample format

[ { "subject": "Physics", "chapter": "Electrostatics", "difficulty_level": "Medium", "stem": "Two charges +2μC and +3μC are separated by 1 m. The electrostatic force between them is approximately:", "option_a": "0.054 N", "option_b": "0.018 N", "option_c": "0.009 N", "option_d": "0.108 N", "correct_option": "A", "explanation": "F = kq₁q₂/r² = 9×10⁹ × (2×10⁻⁶)(3×10⁻⁶) = 0.054 N." } ]
